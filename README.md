# vue 项目模板

前端项目预设模板使用**Vue + Typescript**

## 使用技术

|   名称   |       说明        |
| :------: | :---------------: |
| eslintrc | 代码检查,风格统一 |
| tailwind |    原子化 CSS     |
| naive-ui |     视图组件      |
| prettier |    代码格式化     |

## 插件

|          名称           |          说明          |
| :---------------------: | :--------------------: |
|  unplugin-auto-import   |    API 自动导入插件    |
|     unplugin-icons      | 图标插件,方便使用 icon |
| unplugin-vue-components |    组件自动导入插件    |
